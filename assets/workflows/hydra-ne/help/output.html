Output

The following commands are used to control the type of field and time-history output, plot files, and the intervals at which data is written to each type of file.

Dump

ndump - Output dump files every Ndump (integer, default=0) time steps.

Note: Dump activates checkpoint-restart files, where a restart file will be written every Ndump time steps. In addition, a restart file will be written at the termination of the calculation. A restart file may be read on a subsequent invocation of HYDRA with the "-r" command-line option.

Plot

Options

Options for plot files.

filetype - Specifies the type of plot files, filetype (string, default=serial). Valid values are the following:

  • Serial - This file type is the default for serial calculations. For parallel calculations, this file type requires the serialization of data using a parallel fan-in procedure, but results in a single plot file.
  • Distributed - This file type results in one plot file per processor and is more scalable for problems where a large number of processors are used.

pltype - Specifies the plot file format, pltype (string, default=exodusii). Valid values are the following:

  • ASCII GMV - Generates ASCII GMV files suitable for use with the GMV visualization tool,
  • Exodus-II (CDF) - generates Exodus-II (CDF) format plot files (By default, floating point results are represented with a 32-bit representation and is limited to mesh sizes below approximately 6 x 106 elements),
  • Exodus-II (CDF) 64bit - generates Exodus-II (CDF) format plot files (Floating point results are represented with a 64-bit representation and are limited to mesh sizes below approximately 6 x 106 elements),
  • Exodus-II (CDF) HDF5 - generates Exodus-II (CDF) format plot files (By default, floating point results are represented with a 32-bit representation and allows for larger mesh sizes than the Exodus-II format),
  • Exodus-II (CDF) HDF5 64bit - generates Exodus-II (CDF) format plot files (Floating point results are represented with a 64-bit representation and allows for larger mesh sizes than the Exodus-II format), or
  • ASCII VTK - Generates ASCII VTK format plot files.

Note: The pltype selects the file format used for field output which includes both instantaneous state variables, and time-averaged statistical quantities. At this time, surface field variables associated with model side-sets is only available in the exodusii and exodusii_hdf5 formats.

nplot - Output plot files every Nplot (integer, default=20) time steps.

Plot Variables

Element Variables - Element variables that specifies plot output.

  • cfl - Courant-Friedrichs-Levy number
  • density - Fluid density
  • dist - Normal distance from no-slip/no-penetration surfaces
  • div - Velocity divergence (∇ ⋅ v)
  • enthalpy - Fluid specific enthalpy
  • enstrophy - Enstrophy (ω ⋅ ω/2)
  • helicity - Helicity (v ⋅ ω)
  • pressure - Fluid pressure
  • procid - Processor Id (MPI Rank)
  • temp - Fluid temperature
  • turbeps - Turbulent kinetic energy dissipation rate, (ε)
  • turbke - Turbulent kinetic energy (k)
  • turbnu - Turbulent eddy viscosity
  • vel - Fluid velocity vector
  • vginv2 - Q-criteria, i.e., 2nd invariant of velocity gradient
  • vorticity - Vorticity, i.e., curl of velocity (∇ x v)
  • ystar - y* non-dimensional distance from walls

Node Variables - Node variables that specifies plot output.

  • density - Fluid density
  • dist - Normal distance from no-slip/no-penetration surfaces
  • enthalpy - Fluid specific enthalpy
  • enstrophy - Enstrophy (ω ⋅ ω/2)
  • helicity - Helicity (v ⋅ ω)
  • pressure - Fluid pressure
  • procid - Processor Id (MPI Rank)
  • temp - Fluid temperature
  • turbeps - Turbulent kinetic energy dissipation rate, (ε)
  • turbke - Turbulent kinetic energy (k)
  • turbnu - Turbulent eddy viscosity
  • u - Nodal displacement vector for ALE computations
  • vel - Fluid velocity vector
  • vginv2 - Q-criteria, i.e., 2nd invariant of velocity gradient
  • vorticity - Vorticity, i.e., curl of velocity (∇ x v)

Face Variables - Face variables that specifies plot output.

  • traction - Traction force vector
  • straction - Shear traction force vector
  • ntraction - Normal traction force vector
  • wallshear - Wall shear force
  • yplus - y+ at a wall
  • ystar - y* at a wall
  • varyplus - Spatial variation of y+ at a wall
  • heatflux - Heat flux vector at a wall
  • nheatflux - Normal heat flux at a wall
  • surfarea - Surface area (of each cell) at a wall

ASCII

prtlev - The amount of data to output, prtlev (string, default=param). Valid values are the following:

  • Primary Code Options Only - Suppresses all output except for a data echo of the primary code options,
  • Mesh Coordinates and Topology - Produces data echo of the mesh coordinates and topology, or
  • Primitive Variables - Produces a data echo of the primitive variables every Nprint time steps.

Note: Controls the amount of data written to the ASCII (human-readable) output file.

nprint - Output ASCII files every Nprint (integer, default=10) time steps.

Time-History

Time-History Variables

elementid - Element ID (integer, required).

Element Variables - Element variables that specifies plot output for statistics data.

  • <density> - Mean density
  • density - Fluid density
  • div - Velocity divergence (∇ ⋅ v)
  • enstrophy - Enstrophy, i.e., square of vorticity
  • enthalpy - Fluid specific enthalpy
  • helicity - Helicity, i.e., dot product of velocity and vorticity
  • pressure - Fluid pressure
  • temp - Fluid temperature
  • turbeps - Turbulent dissipation rate (ε)
  • turbke - Turbulent kinetic energy (k)
  • turbnu - Turbulent eddy viscosity
  • vel - Fluid velocity vector
  • vorticity - Vorticity, i.e., curl of velocity (∇ x v)

surfaceid - Surface ID (integer, required).

Surface Variables - Face variables that specifies plot output for statistics data.

  • avgpress - Average pressure for a surface
  • avgtemp - Average temperature for a surface
  • avgvel - Average velocity vector for a surface
  • force - Integral force acting on a surface
  • fvol - Integrated volume of fluid that has crossed a surface
  • heatflow - Heat flow rate crossing a surface
  • massflow - Mass flow rate crossing a surface
  • pressforce - Integral pressure force acting on a surface
  • surfarea - Surface area
  • viscforce - Integral viscous shear force acting on a surface
  • volumeflow - Volume flow rate crossing a surface

Frequency

nstep - Output Time-History data every Nstep (integer, default=1) time steps.

Velocity Min/Max

nminmax - Output min/max values of velocity every Nminmax (integer, default=10) time steps.

Note: Set the interval to report the min/max values of the velocity to Nminmax. The min/max values are normally written to the screen at run-time.

Statistics

Options for accumulating statistics of fluctuating variables.

starttime - Start time for collecting statistics, tstart (float, default=0.0).

endtime - End time for collecting statistics, tend (float, default=1.0).

plotwinsize - Time window size for output of field statistics, size (float, default=0.1).

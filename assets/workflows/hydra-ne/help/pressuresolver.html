Pressure Solver

Define the attributes of the pressure-Poisson solver.

type - Specifies the preconditioner - Krylov solver combination, type (string, default=AMG). Values can be one of the following:

  • AMG - Algebraic multigrid with the conjugate gradient method,
  • SSORCG - Successive over-relaxation preconditioner with the conjugate gradient method, or
  • JPCG - Jacobi preconditioner with the conjugate gradient method.

amgpc - Specifies the preconditioner package for the AMG solver, amgpc (string, default=ML). Only effective if type=AMG. Values can be one of the following:

  • ML - Multilevel preconditioner package, or
  • HYPRE - Hypre preconditioner package.

hypre_type - Specifies the preconditioner sub-package for the Hypre/AMG solver, hypre_type (string, default=BOOMERAMG). Only effective if type=AMG and amgpc=HYPRE. Values can be one of the following:

  • BOOMERAMG - BoomerAMG preconditioner.

cycle - Specifies the type of AMG cycle, cycle (string, default=V). Values may be one of the following:

  • V - Use a V-cycle, or
  • W - Use a W-cycle.

solver - Specifies the underlying Krylov solver to be used with AMG, solver (string, default=CG). Values can be one of the following:

  • CG - Conjugate gradient method,
  • BCGS - Stabilized bi-conjugate gradient squared method, or
  • FGMRES - Flexible generalized minimum residual method.

pre_smooth - Set the number of pre-smoothing sweeps for AMG, pre_smooth (integer, default=1).

post_smooth - Set the number of post-smoothing sweeps for AMG, pre_smooth (integer, default=1).

levels - Set the maximum number of AMG levels to use in the multigrid cycle, levels (integer, default=20).

nitmax - Set the maximum number of iterations, Nitmax (int, default=500). In the case of AMG, this is the maximum number of V or W cycles.

nitchk - Set the number of iterations to take before checking convergence criteria, Nitchk (int, default=2).

eps - Define the convergence criteria for linear solver, ε (float, default=1.0E-5).

pivot - Specify the value of a zero pivot for preconditioner, pivot (float, default=1.0E-16).

convergence - Enable/disable the convergence metrics for the linear solver, convergence (string, default=Off).

  • On - Activate convergence output, or
  • Off - Suppress convergence output.

diagnostics - Enable/disable the diagnostic information from the linear solver, diagnostics (string, default=Off).

  • On - Activate diagnostic output, or
  • Off - Suppress diagnostic output.

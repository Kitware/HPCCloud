Heat Sources

The specification of various heat sources for the momentum equations.

name - A descriptive heat source name (string, default=null). name is not used in the derived control file. A unique id for each specific heat source is generated underneath the cover of the web interface. While providing a descriptive name is not required, these descriptive names clearly increase usability.

type - The type of heat source (string, default=Heat Source). The type must be one of the following:

  • Heat Source - Specifies a volumetric heat source.

Note: Specifies a volumetric heat source. It may be time-varying, by specifying a load curve. In addition, the heat source may be prescribed for a specific element set (element block) or all element sets in the mesh.

Heat Source

Specifies a volumetric heat source. It may be time-varying, by specifying a load curve. In addition, the heat source may be prescribed for a specific element set (element block) or all element sets in the mesh.

setids - One (or more) node set(s) on which the heat source will be applied, setId (integer, required).

load_curve

A load curve allows for the specification of time-dependent body forces. The load curve is defined as a list of time and factor pairs:

  • ti The time value
  • vi The load factor

name - A descriptive load curve name (string, default=null). name is not used in the derived control file. A unique id for each specific load curve is generated underneath the cover of the web interface. While providing a descriptive name is not required, these descriptive names clearly increase usability.
